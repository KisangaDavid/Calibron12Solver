Solver for the famous Calibron 12 puzzle and others like it. Utilizes recursive backtracking and pruning to greatly cut down on the search space and find a solution quickly. The c++ version is pretty optimized - finding a solution to the Calibron 12 puzzle takes ~15 milliseconds on my i5-1334U 1.3 GHz laptop.
